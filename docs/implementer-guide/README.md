# Implementer Guide

This guide is for program managers and NGO staff who are setting up and running CareLoom for a community health worker (CHW) home visiting program. No coding experience is required.

## Sections

| Guide | What You Will Learn |
|-------|---------------------|
| [First Setup](first-setup.md) | Creating your organization, adding admin accounts, setting up CHW accounts, and uploading curriculum content after deployment |
| [Curriculum Management](curriculum-management.md) | Building curricula with stages, modules, components, and scheduling visits to baby ages |
| [Managing CHWs](managing-chws.md) | Adding community health workers, assigning supervisors, and managing their accounts |
| [Monitoring Visits](monitoring-visits.md) | Tracking visit completion, viewing reports, and exporting data from the admin dashboard |
| [Security](security.md) | Changing default passwords, setting up HTTPS, backups, and access control |
| [Localization](localization.md) | How language support works and how to request new languages |

## Before You Start

Make sure CareLoom is already deployed and running. If you have not deployed yet, see the [Getting Started](../getting-started/self-hosted-deploy.md) guide first.

You will need:
- The URL of your admin dashboard (e.g., `https://admin.yourorg.example.com`)
- An admin username and password (default: `admin` / `admin`)

## How This Guide Is Organized

Each section is self-contained. You can read them in order for initial setup, or jump to a specific topic when you need it later. Screenshots reference the admin dashboard interface at `http://localhost:3000` but your URL may differ.
