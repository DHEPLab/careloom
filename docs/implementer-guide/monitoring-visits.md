# Monitoring Visits

The admin dashboard provides tools to track visit completion, view reports, and export data for analysis.

## Visit Lifecycle

Each visit moves through these statuses:

| Status | Meaning |
|--------|---------|
| **Scheduled** | The visit has been automatically generated based on the curriculum schedule and the baby's age |
| **Started** | The CHW has opened the visit in the mobile app |
| **Completed** | The CHW has finished all modules and submitted the visit |
| **Expired** | The visit window passed without the CHW completing the visit |

## Viewing Visit Status

1. In the admin dashboard, click **Babies** in the left sidebar.
2. Select a family to see all scheduled, completed, and pending visits for that baby.
3. Each visit shows:
   - The lesson (module) name
   - The assigned CHW
   - The scheduled date
   - The current status
   - Completion time (if completed)

## Viewing Reports

Reports give you an overview of program activity across all CHWs and families.

1. Click on a report section in the dashboard.
2. You can view:
   - **Visit completion rates** -- what percentage of scheduled visits have been completed
   - **CHW performance** -- visits completed per CHW, broken down by time period
   - **Family coverage** -- how many families have received visits on schedule
   - **Overdue visits** -- visits that are past their scheduled window and still incomplete

## Exporting Data

CareLoom supports exporting visit and family data for external analysis.

1. Navigate to the data view you want to export.
2. Click the **Export** button.
3. The system generates an Excel file (.xlsx) containing the selected data.
4. Download the file to your computer.

Exported data can include:
- Visit records with completion times and CHW assignments
- Baby and carer demographic information
- Questionnaire responses
- Visit position (GPS) data if location tracking is enabled

## Using Visit Position Data

If GPS tracking is enabled, each visit records the CHW's location at the time of the visit. This can be used to:

- Verify that visits are being conducted at the family's home
- Map coverage areas
- Identify areas that are underserved

Position data is visible in the exported data and in the visit detail view.

## Understanding Questionnaire Responses

If your curriculum includes questionnaires, responses are saved with each visit. To review them:

1. Open a completed visit from the baby's visit list.
2. View the questionnaire responses in the visit detail.
3. Export questionnaire data for aggregate analysis.

## Tips

- **Check weekly.** Review visit completion rates at least weekly to identify issues early.
- **Follow up on expired visits.** If visits are expiring frequently, investigate whether CHWs need additional training, transportation support, or schedule adjustments.
- **Compare across CHWs.** Differences in completion rates between CHWs may indicate workload imbalances or areas where additional support is needed.
- **Export regularly.** Keep regular data exports as backups and for longitudinal analysis.
